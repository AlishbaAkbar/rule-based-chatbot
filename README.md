
# 🤖 Rule-Based AI Chatbot

**Project 1 | DecodeLabs AI Internship — Batch 2026**

## About
A rule-based chatbot built with Python and Tkinter that responds
to predefined user inputs using dictionary-based O(1) lookup logic.

## Features
- 3-tier intent matching (dynamic → exact → keyword)
- GUI built with Tkinter
- Input sanitization
- Randomized responses for variety

## How to Run
```bash
python chatbot.py
```

## Tech Stack
- Python 3
- Tkinter (GUI)
